HTML Tags & Attributes

HTML ELements and Attributes
Tag Name Description
<h1> - <h6> Heading element HTML uses 6 differnet levels of heading to order from highest to lowest.
<video> Video Video tag used to embed video player and requires the src attribute.
<em> Emphasis The element emphasises text and will be italicized by default.
HTML Table tags and Attributes
Tag Name Description
<tr> Table Row Element The table row element is used when adding rows to a table.
<td> Table data element The table data element can be nested inside a table row element to add cell to a table.
colspan colspan Attribute Indicates how many columns the cell should span within the table.
